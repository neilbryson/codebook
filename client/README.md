# Codebook client

## Requirements

* [Node.js](https://nodejs.org/en/)

## Installation

```bash
npm install
```

## Starting the development server

```bash
npm start
```

## Build bundle

```bash
npm run build
```

## Not familiar with TypeScript?

Switch to the [`js` branch](https://github.com/neilbryson/codebook/tree/js) to
use the JavaScript version.
